## 🏗️ Featured Project: Real-Time BTC Data Pipeline
A fully containerized microservice architecture that automates the ingestion of live market data from the Binance API into a persistent PostgreSQL database.

### 🛠️ The Tech Stack
*   **Logic:** Python (Requests, Psycopg2)
*   **Containerization:** Docker & Docker Compose
*   **Database:** PostgreSQL (Relational Modeling)
*   **Infrastructure:** Linux (Ubuntu 24.04 LTS on Dell XPS 13)

### 🧠 Engineering Wins
*   **Modular Architecture:** Used Docker Compose to decouple the database from the logic, allowing for "one-command" deployment.
*   **Persistence:** Built a relational schema to handle time-series financial data.
*   **Resilience:** Implemented custom error-handling (`try/except`) to manage API rate limits and network latency.
